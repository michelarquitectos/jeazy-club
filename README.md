# Jeazy Club

Sitio web estático de Jeazy Club, preparado para publicarse con GitHub Pages.

## Contenido

- `index.html`: estructura, contenido, SEO básico, registro y ventanas emergentes.
- `styles.css`: sistema visual y diseño responsive.
- `script.js`: validación de edad, promoción, cookies, menú y formulario.
- `assets/jeazy-club-logo.png`: logotipo oficial de esta marca.
- `robots.txt` y `sitemap.xml`: base para buscadores.
- `.nojekyll`: evita que GitHub Pages procese el sitio como Jekyll.

## Publicar en GitHub Pages

1. Crea un repositorio nuevo en GitHub.
2. Sube **todo el contenido de esta carpeta** a la rama `main`.
3. Abre `Settings` → `Pages`.
4. En `Build and deployment`, elige `Deploy from a branch`.
5. Selecciona la rama `main` y la carpeta `/root`.
6. Guarda. GitHub mostrará la dirección pública después de unos minutos.

## Datos que debes reemplazar

Busca y actualiza en `index.html`:

- `https://wa.me/` por el enlace completo de WhatsApp.
- `hola@jeazyclub.mx` por el correo oficial.
- Textos y enlaces del aviso de privacidad, cookies y términos.
- Productos, beneficios y condiciones definitivas de la membresía.

Actualiza también la dirección provisional de `sitemap.xml` cuando conozcas el dominio final.

## Importante sobre el registro

El formulario actual es una demostración visual y **no envía ni almacena datos**. Antes de recibir fotografías del INE o firmas digitales se necesita:

- servidor privado con HTTPS;
- almacenamiento cifrado;
- controles de acceso y registro de actividad;
- política de conservación y eliminación;
- aviso de privacidad revisado legalmente;
- protección contra abuso y automatizaciones;
- proveedor de firma electrónica adecuado al documento.

No conectes el formulario a correo electrónico ni a servicios públicos de formularios para recibir identificaciones oficiales.

## Vista local

Puedes abrir `index.html` directamente. Algunas funciones funcionan mejor usando un servidor local sencillo.
